Overview

This assignment helps you understand modern HTML5 structure while applying key concepts of web accessibility and search engine optimization (SEO). The goal is to create a simple web page that is easy to navigate, readable, and optimized for both humans and machines.

Objective

Build a basic web page using semantic HTML5 tags that follow accessibility best practices and beginner-level SEO principles. Your layout should be well-structured, screen-reader friendly, and easy for search engines to understand.

Guidelines

  • Use semantic HTML5 elements such as <header>, <nav>, <main>, and <footer>.
  • Avoid using <div> or <span> unless absolutely necessary.
  • Apply accessibility features like labels, alt text, and proper heading hierarchy.
  • Ensure your page content is descriptive, readable, and SEO-friendly.

Deliverables

Your submission should include:

  1. A single HTML file named index.html.
  2. Semantic structure using appropriate HTML5 elements.
  3. Clear headings following a logical hierarchy.
  4. Accessibility enhancements using the right tags and attributes.
  5. SEO-friendly metadata and content.

Tips

  • Use semantic tags meaningfully to represent content.
  • Ensure your structure is accessible to screen readers.
  • Include descriptive titles, headings, and meta descriptions for SEO.
  • Keep your HTML clean, readable, and well-indented.
